# 🛍 Trendique – MERN E-commerce Website

Trendique is a full-stack e-commerce web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application allows users to browse products, manage carts, and experience a seamless online shopping interface.

---

## 🚀 Features

- User authentication (Login / Signup)
- Product listing page
- Add to cart functionality
- Responsive UI design
- REST API integration
- MongoDB database storage
